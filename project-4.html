Queue Coding Project

Project Overview

This project was given by my coding supervisors to test my understanding of Python. Inspired by my experience working at NECTEC (National Electronics and Computer Technology Center) in a 3D body scanning lab, where an average of 20-30 people visited daily to have their bodies measured at various stations. I observed that managing such a large number of participants became challenging. The staff found it difficult to track each participant's progress, complicating the process and causing unnecessary delays.

This is where I devised a simple solution after learning about QueueManager and combining it with fundamental Python concepts such as if-else statements, loops, and lists. This project has greatly inspired me to enhance my coding skills because I recognize areas for improvement. While my Python skills are still limited, I am eager to learn and improve further.
